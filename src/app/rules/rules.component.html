REGLES

Il y a un deck de 100 cartes mélangées

Le joueur débute le jeu
> une première carte (carte 1) avec une date révélée est placée sur la timeline.
> le joueur pioche une carte (carte 2)
> le timer démarre (5min, décroissant)

Le joueur place la carte 2 avant ou après la carte 1

Le joueur valide son choix :
la date de la carte 2 est révélée
le jeu valide ou non la position

Si la position est fausse :
- une pénalité du temps de 2 secondes est réalisée sur le timer
- la carte révélée disparaît et n’est plus dans la pioche
Si la position est validée :
- la carte reste sur la timeline
- le joueur peut piocher une autre carte

Le joueur pioche une autre carte (carte 3) et la place entre les 2 cartes qui lui semblent le plus pertinent

Le jeu se termine quand le chrono est terminé ou quand toutes les cartes ont été posées

Le joueur voit son score (nombre de cartes bien placées en temps imparti)

Le joueur peut rejouer
